# DreamTeam — Bootstrap Rules (for empty project)

When this file is present, the project uses DreamTeam Autonomous Development System.

## First step: Deploy environment

If .dreamteam/ does NOT exist, run:
```
python -m dreamteam new-project .
```

## All commands

Use `python -m dreamteam <command>` (works without PATH). Commands:
- python -m dreamteam new-project <path>  — Create project
- python -m dreamteam run-next   — Get next task
- python -m dreamteam sync-tasks          — Sync tasks to DB
- python -m dreamteam verify-tasks        — Verify consistency
- python -m dreamteam update-task <id> <status>
- python -m dreamteam task-counter
- python -m dreamteam recover             — Recovery from failures
- python -m dreamteam check-memory
- python -m dreamteam current-project

## After deployment

Load rules from .cursor/rules/ and agents from .cursor/agents/
